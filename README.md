# Hangman Game 🎮

A command-line Hangman game built in Python as part of the CodeAlpha Python Programming Internship.

## How to Play

1. Run `python hangman.py`
2. Choose a difficulty level:

   * Easy (8 wrong guesses)
   * Medium (6 wrong guesses)
   * Hard (4 wrong guesses)
3. Guess one letter at a time.
4. Reveal the hidden word before running out of attempts.
5. View your session statistics after finishing.

## Features

* Three difficulty levels
* Random word selection
* ASCII-art hangman visualization
* Input validation and error handling
* Session statistics tracking
* Replay option after each round
* Object-oriented design using a `GameStats` class

## Requirements

* Python 3.7+

## Run

```bash
python hangman.py
```
