# Hangman Game 🎮

A command-line Hangman game built in Python as part of the CodeAlpha Python Programming Internship.

## Run

```bash
python hangman.py
```

## How to Play

1. Choose a difficulty level.
2. Guess one letter at a time.
3. Reveal the hidden word before running out of attempts.
4. View your session statistics after finishing.

## Features

* Three difficulty levels
* Random word selection
* ASCII-art hangman visualization
* Input validation and error handling
* Session statistics tracking
* Replay option after each round
* Object-oriented design using a `GameStats` class

## Requirements

* Python 3.7+

## Skills Demonstrated

* Python Fundamentals
* Object-Oriented Programming (OOP)
* Data Structures
* Input Validation
* Game State Management
